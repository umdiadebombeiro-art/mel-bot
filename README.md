# Instagram @sargentomellocasal → Telegram (seguro, 100% grátis)

Posta normal, com 3 travas anti-ban:
- roda a cada 30 min
- máx 5 posts por vez
- 3 segundos entre mensagens
- legenda com fonte

Como usar:
1. Crie bot no @BotFather → copie token
2. Adicione bot como admin no canal
3. Faça upload destes arquivos num repositório público no GitHub
4. Settings → Secrets → Actions:
   - BOT_TOKEN = seu token
   - CHAT_ID = @seucanal
5. Ative Actions
